# Security Policy

## Supported Versions

This is an academic project. Only the latest commit on `main` is actively maintained.

| Version | Supported |
|---------|-----------|
| latest (main) | Yes |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public GitHub issue.

Instead, report it by emailing: **terzi.diego02@gmail.com**

Please include:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact

You can expect an acknowledgement within 7 days. This is a student project, so
response times may vary during academic periods.
